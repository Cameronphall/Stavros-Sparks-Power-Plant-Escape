# Stavros Sparks: Power Plant Escape

Stavros is an electrician who finds himself trapped inside a malfunctioning power plant. Solve puzzles, collect batteries, and restore power to escape before time runs out. The game is centered around math and logic-based puzzles.

This project is a complete, published game available for download on itch.io.

---

## Features

- Custom 2D animations
- Finite state machine (FSM) for game state management
- Custom 2D sprite rendering
- Puzzle-based level progression
- Packaged executable that runs without additional installations

---

## Play the Game

Download the published version here:  
https://cameronhall193.itch.io/stavros-sparks-power-plant-escape

To play:
1. Download the zip file from itch.io  
2. Extract all files  
3. Run the `.exe` file

---

## Controls

- **WASD / Arrow Keys** — Move
- **F** — Interact
- **ESC** — Menu

---

## Technologies Used

- Go
- Raylib-Go (Go bindings for Raylib)
- Custom game loop and rendering pipeline

---

## What I Learned

Building this project taught me how to structure a larger interactive codebase, manage game state, handle asset pipelines, and package a standalone executable for distribution.
